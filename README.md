# 🏠 House Price Prediction

This project uses Machine Learning to predict house prices based on features like area, bedrooms, bathrooms, etc.

## 📊 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Project Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model training (Linear Regression)
- Model evaluation using RMSE and R2 Score

## 📁 Files
- `House Prices Case Study.ipynb` → Main project notebook

## 🚀 How to Run
1. Open the notebook in Jupyter or Google Colab  
2. Run all cells  

## 📌 Goal
To build a basic regression model for real-world price prediction
